Offline DB

An offline DB demo based on AngularJS framework and HTML5 IndexedDB

Download this project as a .zip file Download this project as a tar.gz file

This is a single page application based on AngularJS framework and HTML5 IndexedDB. It manages a simple local database, with which users can add, remove, search and list records.

One tip: All the photos must be kept in photos/ directory. When you create a new slide, only images under that directory can be selected. After all, it's just a html app, isn't it?
